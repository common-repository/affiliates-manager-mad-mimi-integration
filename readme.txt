An addon for the Affiliates Manager plugin to sign up affiliates to your Mad Mimi list

== Description ==

This addon allows you to specify a Mad Mimi list name in the addon settings. When affiliates join your site, they get signed up to the specified Mad Mimi list automatically.

This addon requires the [Affiliates Manager Plugin](https://wordpress.org/plugins/affiliates-manager/).

After you install this addon, go to the Mad Mimi settings interface and specify your API Key and the list name.

Read the following page for step by step usage documentation:
https://wpaffiliatemanager.com/signup-affiliates-to-your-madmimi-list/

== Installation ==

1. Upload `affiliatemgr-madmimi-integration` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
